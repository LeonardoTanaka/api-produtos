# Sistema de Monitoramento de Produtos

## 🚀 Funcionalidades
- Cadastro de produtos
- Armazenamento em banco PostgreSQL
- Geração de relatório em Excel
- Envio automático por email

## 🛠️ Tecnologias
- FastAPI
- PostgreSQL
- Pandas
- SMTP (email)

## ▶️ Como rodar
pip install -r requirements.txt
uvicorn main:app --reload
